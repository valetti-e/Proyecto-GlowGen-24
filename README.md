# GlowGen 24

Proyecto web desarrollado como trabajo final, enfocado en la presentación de servicios de estética y bienestar. La propuesta combina diseño visual moderno con una estructura clara, priorizando la experiencia del usuario (UX) y la accesibilidad.

---

## Demo online

https://valetti-e.github.io/Proyecto-GlowGen-24/

El sitio se encuentra desplegado utilizando GitHub Pages, lo que permite su acceso público sin necesidad de instalación.

---

## Objetivo del proyecto

El objetivo principal fue diseñar y desarrollar una landing page funcional que:

* Presente servicios de estética de forma clara y atractiva
* Mejore la experiencia del usuario mediante navegación intuitiva
* Aplique buenas prácticas de diseño UX/UI
* Sea adaptable a distintos dispositivos (responsive design)

---

## Enfoque UX/UI

Durante el desarrollo se tuvieron en cuenta los siguientes aspectos:

* Jerarquía visual para destacar información importante
* Uso de colores asociados al bienestar y la estética
* Organización del contenido en secciones claras
* Llamados a la acción (CTA) para incentivar la interacción
* Diseño simple y limpio para mejorar la usabilidad

---

## Tecnologías utilizadas

* HTML5: estructura del sitio
* CSS3: estilos personalizados
* Bootstrap: sistema de grillas y componentes responsive
* Git: control de versiones
* GitHub: almacenamiento del repositorio y despliegue

---

## Diseño responsive

El sitio fue desarrollado para adaptarse a diferentes resoluciones de pantalla:

* Dispositivos móviles
* Tablets
* Escritorio

Se utilizaron media queries y herramientas de Bootstrap para lograr la adaptabilidad.

## Instalación y uso

Para ejecutar el proyecto de manera local:

1. Clonar el repositorio:

```
git clone https://github.com/valetti-e/Proyecto-GlowGen-24.git
```

2. Acceder a la carpeta del proyecto

3. Abrir el archivo index.html en cualquier navegador web

No requiere instalación de dependencias adicionales.
